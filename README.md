
# TweetX_Server

Express server for TweetX 


## Demo

server https://tweetxserver.onrender.com


## Run Locally

Clone the project

```bash
  git clone https://github.com/Csb-218/TweetX_Server.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon
```
